# Engine Controls
**[⬅ Back to Main README](../README.md)**

The engine is designed to be input-agnostic and supports both **QWERTY** and **AZERTY** layouts automatically.

## Camera & Movement
| Action | Primary Keys | Alternative Keys |
| :--- | :--- | :--- |
| **Move Forward** | `W` | `Z` |
| **Move Backward** | `S` | `S` |
| **Strafe Left** | `A` | `Q` |
| **Strafe Right** | `D` | `D` |
| **Rotate View Left (Yaw)** | `Q` | `A` |
| **Rotate View Right (Yaw)** | `E` | `E` |

## Object Manipulation
| Action | Keys | Context |
| :--- | :--- | :--- |
| **Rotate Object X/Y** | `Arrow Keys` | Rotates the model in world space |

## System
| Action | Key |
| :--- | :--- |
| **Quit Application** | `ESC` |
